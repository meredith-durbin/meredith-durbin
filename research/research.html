My Research

WFC3
  • Currently working on updating the WFC3/IR bad pixel table
  • Built web interface to search, view, & analyze >170,000 WFC3 in-flight images
  • Characterized impact of WFC3/IR ''blobs'' on stellar photometry (WFC3 ISR 2015-06)
  • Studied properties of WFC3/IR ''snowballs'' (WFC3 ISR 2015-01)
  • PI of Cycle 23 calibration programs 14367 (UVIS bowtie monitor) and 14374 (IR dark monitor)
  • Created HST photometric conversion web tool
CANDELS
  • Performed artificial galaxy tests on all CANDELS fields to characterize completeness in magnitude/radius space
    • Incorporated results into Py2PAC angular correlation function code
    • Results used by Cohen et al. (2016, in prep) to determine observability of mock galaxies
  • Performed pair tests of COSMOS photometric redshifts to determine core photometric redshift error (Nayyeri et al. 2016, in prep)
  • Created interactive tool to search for OIII and Hα emission line galaxies in GOODS fields
